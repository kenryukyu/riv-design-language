# Vertical Spacing

## Variable Names:

* vertical-spacing-xxs
* vertical-spacing-xs
* **vertical-spacing-sm**
* **vertical-spacing-md \(default\)**
* **vertical-spacing-lg**
* vertical-spacing-xl
* vertical-spacing-xxl



## Example Variable Settings

* vertical-spacing-small = 8px
* vertical-spacing-medium = 16px
* vertical-spacing-large = 24px
