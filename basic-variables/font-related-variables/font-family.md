# Font Family

## Variable Names:

* **font-family-base **\(default\)
* font-family-alt-1
* font-family-alt-2
* font-family-alt-3

## Example Variable Settings:

* font-family-base = 'Hiragino Kaku Gothic ProN, Arial, "メイリオ", Meiryo, sans-serif'
* font-family-alt-1 = 'Roboto'
* font-family-alt-2 = 'Lobster'
