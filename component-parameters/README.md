description: >-
  Component sizing language used for component elements.  There are 8 total
  sizes you can specify, however most components will only have three main
  sizes, "small", "medium" (default), and "large".
---

# Component Parameters

## Variable Name: 

  size

## Possible Values:

* mini
* tiny
* small
* medium \(default\)
* large
* big
* huge
* massive
